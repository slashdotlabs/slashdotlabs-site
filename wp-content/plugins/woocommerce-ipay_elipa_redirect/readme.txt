It's a WooCommerce extension allowing clients make payment with iPay.

== Description ==
WooCommerce payment gateway plugin for iPay which accept payments from client through M-Pesa, Airtel, Equitel, VISA, Mastercard and Pesalink. Fully supported with WordPress and WooCommerce latest versions. iPay latest API 3.0 compatible.

= Features =
* Enable and disable various payment channels.
* Enable/Disable Test Mode

= How to Setup Account with iPay = 
[Step by step iPay Account Setup Guide](https://ipayafrica.com/for-business/)

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. After activation, you can set options from `WooCommerce -> Settings -> iPay` menu
